📌 Agile Product Planning – Portfolio Project

This repository showcases my Agile product planning workflow for a digital product concept.
The focus of this project is translating ideas into structured development tasks using Agile methodology.

🚀 Tools Used

🎨 Figma – UI/UX design and interface prototyping

🗂 Notion – Project management and backlog organization (in PDF format)

📂 Project Structure

This repository contains four main documents:

File	Description
Epic Story / Story Mapping	High-level product vision and feature hierarchy (Epic → User Story → Task).
Product Backlog	List of prioritized product features and improvements.
Sprint Backlog	Tasks selected for development during a sprint.
Development Backlog	Detailed breakdown of technical and implementation tasks.

🧠 Agile Approach

This project demonstrates how I:

Break down product ideas into Epic → User Story → Task

Organize work into structured backlogs

Align UI design (Figma) with project planning (Notion)

Apply Agile thinking for product development

🎯 Goal

To showcase my ability to plan and structure a digital product workflow before development begins, ensuring clear priorities and efficient team collaboration.
